# 1.2.0.3 Pre Registration

**Release Version:** v1.2.0.3

**Release Type:** Patch Release

**Release Date:** 5th September, 2025

#### **Overview** <a href="#overview" id="overview"></a>

This patch release focuses on critical fixes to the automation test suites for the Pre-Registration repository. The updates address issues in test scripts to establish a zero-failure baseline, enhancing the stability and reliability of testing across these identity management components.

## **Major Highlights/Features** <a href="#major-highlights-features" id="major-highlights-features"></a>

* Fixed: Automation test `Prereg_GetPRIDByDateRange_All_Valid_Smoke_sid` now uses a valid registrationCenterId with appointments in the Pre-Registration DB.&#x20;
