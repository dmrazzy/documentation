# 1.2.0.3

**Release Version**: 1.2.0.3

**Release Type:** Patch Release

**Release Date**: 5th September, 2025

&#x20;

### **Overview:** <a href="#overview" id="overview"></a>

The MOSIP 1.2.0.3 release is a patch update focused on enhancing the platform's performance and resolving critical bugs to ensure robust functionality and reliability in real-world identity management deployments. This release builds on the 1.2.0.x series, addressing issues that impact field operations and improving system efficiency for large-scale use cases.

### Major Areas of Work <a href="#major-areas-of-work" id="major-areas-of-work"></a>

* **Performance Optimization**: System-wide enhancements to improve scalability and responsiveness.
* **Core Modules**: Bug fixes targeting critical functionality in registration, and data processing workflows.
