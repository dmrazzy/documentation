### **Overview**

We are pleased to announce **API Test Commons v1.4.1**, a patch release focused on improving deployment compatibility for functional test executions in containerized environments.

This release introduces necessary Helm configuration updates to support integer-based container user configuration required for newer Inji organization deployments. The update ensures seamless execution of API test rigs when deployed using updated Helm charts sourced from the MOSIP Functional Test repository.

Version **1.4.1** builds upon **v1.4.0**, providing stability and environment compatibility improvements without functional changes to existing automation workflows.
