# v1.2.2.0 DSL

**Release Name**: DSL Orchestrator & Packet Creator

**Release Version**: 1.2.2.0

**Release Type**: Minor Release

**Release Date**: 20th April, 2026

### Overview

We are pleased to announce **DSL Orchestrator & Packet Creator v1.2.2.0**, released to support environments running on **Java 11**.

This release contains functionality equivalent to [**v1.4.0 DSL**](v1.4.0-dsl.md), with the primary difference being **Java runtime compatibility**. While **v1.4.0 is built for Java 21**, **v1.2.2.0 is provided specifically for Java 11 environments** where upgrading Java versions is not yet feasible.

All DSL scenarios, reporting improvements, bug fixes, and enhancements available in [**v1.4.0 DSL**](v1.4.0-dsl.md) are also applicable to this release unless otherwise specified.
