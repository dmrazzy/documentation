# Media Kit

Use the official MOSIP media kit when you refer to MOSIP in public material.

MOSIP assets need consistent use. This helps partners, governments, contributors, and media teams represent MOSIP correctly.

### Get the official assets

Access the official MOSIP media kit and brand guidelines here:

* [MOSIP Media Kit and Brand Guidelines](https://www.mosip.io/resource_centre?resource=media-kit-brand-guidelines)

The logo kit is available within that section on the MOSIP website.

{% endhint %}
